# BLUE Estimator and Minimum Variance Portfolio

This repository contains coursework from the *Matrices in Economics* module, where key econometric and portfolio optimisation concepts were implemented using matrix algebra.

The original project was completed in **MATLAB**, and later reimplemented in **Python** to demonstrate the same concepts using modern data analysis tools.

---

## Project Objectives

The project focuses on two core applications of matrix methods in economics and finance:

### 1. BLUE Estimator (Best Linear Unbiased Estimator)

The project demonstrates how the OLS estimator can be derived using matrix algebra.

Key concepts explored include:

- Matrix representation of the linear regression model
- Properties of the BLUE estimator
- Variance-covariance matrix of estimators
  
---

### 2. Minimum Variance Portfolio

Using portfolio theory, the project constructs the **minimum variance portfolio** using matrix methods.

Key steps include:

- Constructing the covariance matrix of asset returns
- Applying matrix inversion to derive optimal weights
- Solving the constrained optimisation problem
- Interpreting the resulting portfolio allocation
